# 停车场管理系统 java1083

## 项目概述

停车场管理系统是一个结合了车位预约管理以及停车场推荐的综合性系统。基于Python编程语言，采用Django框架进行开发，实现了B/S架构的全方位解决方案。

## 技术栈

- **后端**: Django框架
- **数据库**: MySQL数据库集群
- **车牌识别**: 内嵌算法，支持所有国内车牌

## 功能模块

- 车位预约管理
- 停车场推荐
- 车牌识别

## 系统角色

- 管理员：负责系统维护与车位管理
- 用户：进行车位预约与停车场选择

## 运行环境

- Python运行环境
- 支持MySQL的数据库环境

## 部署步骤

1. 环境准备：确保Python和MySQL环境已安装
2. 数据库配置：根据集群需求配置MySQL数据库
3. 系统部署：部署后端服务与前端界面
4. 系统启动：运行系统服务
